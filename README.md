# 외행성의 순행과 역행 시뮬레이션

지구에서 관측한 외행성(화성)의 겉보기 운동을 시각화하는 교육용 웹 시뮬레이션입니다.

## 사용 방법

### 실행하기
`index.html` 파일을 웹 브라우저에서 열면 바로 실행됩니다. (별도 서버 불필요)

### 화면 구성

| 영역 | 설명 |
|------|------|
| 태양 중심 뷰 | 실제 태양계에서 지구와 화성의 공전 궤도를 보여줍니다 |
| 지구 중심 뷰 | 지구를 중심으로 태양과 화성의 겉보기 운동을 보여줍니다 |
| 천구 뷰 | 하늘에서 화성이 별들 사이를 어떻게 이동하는지 보여줍니다 |

### 컨트롤 버튼

| 버튼 | 기능 |
|------|------|
| ▶ 재생 | 애니메이션을 시작합니다 |
| ⏸ 일시정지 | 현재 위치에서 애니메이션을 멈춥니다 |
| ⏹ 정지 | 애니메이션을 멈추고 처음(합 위치)으로 돌아갑니다 |

### 슬라이더 사용법

- **0% (왼쪽 끝)**: 합(Conjunction) - 태양-지구-화성이 일직선
- **100% (오른쪽 끝)**: 다음 합 (1회합주기 = 약 780일 경과)
- 슬라이더를 드래그하여 원하는 시점의 행성 위치를 확인할 수 있습니다
- 역행이 일어나는 구간을 직접 찾아볼 수 있습니다

### 정보 패널 읽기

| 항목 | 설명 |
|------|------|
| 현재 운동 상태 | 순행(녹색), 역행(빨간색), 유(노란색) 표시 |
| 회합주기 진행 | 합에서 시작하여 경과한 일수 |
| 외행성 이각 | 태양과 화성 사이의 각도 |
| 외행성 위치 | 현재 화성의 위치 설명 |

## 학습 포인트

### 순행 (Prograde)
- 화성이 별들 사이를 **서 → 동** 방향으로 이동
- 평소 대부분의 시간 동안 순행

### 역행 (Retrograde)
- 화성이 별들 사이를 **동 → 서** 방향으로 이동
- 지구가 화성을 추월할 때 발생하는 겉보기 현상

### 합 (Conjunction)
- 태양-지구-화성이 일직선 상에 위치
- 시뮬레이션의 시작점과 끝점

## 수업 활용 예시

1. **역행 구간 찾기**: 슬라이더를 움직여 역행이 시작되고 끝나는 지점을 찾아보세요
2. **천구 궤적 관찰**: 재생 버튼을 눌러 화성이 하늘에서 그리는 궤적을 확인하세요
3. **두 뷰 비교**: 태양 중심 뷰와 지구 중심 뷰를 비교하여 왜 역행이 발생하는지 이해하세요

## 기술 정보

- 순수 HTML/CSS/JavaScript로 구현
- 외부 라이브러리 없음
- 화성 공전주기: 687일
- 지구 공전주기: 365.25일
- 회합주기: 약 780일
