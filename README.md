# 행성의 순행과 역행 시뮬레이션

지구에서 관측한 행성의 겉보기 운동을 시각화하는 교육용 웹 시뮬레이션입니다.
외행성(화성)과 내행성(금성) 두 가지 시뮬레이션을 제공합니다.

## 사용 방법

### 실행하기
`index.html` 파일을 웹 브라우저에서 열면 바로 실행됩니다. (별도 서버 불필요)

### 파일 구조

| 파일 | 설명 |
|------|------|
| `index.html` | 메인 선택 페이지 - 외행성/내행성 선택 |
| `outer.html` | 외행성(화성) 순행/역행 시뮬레이션 |
| `inner.html` | 내행성(금성) 순행/역행 시뮬레이션 |

### 화면 구성

| 영역 | 설명 |
|------|------|
| 태양 중심 뷰 | 실제 태양계에서 행성들의 공전 궤도를 보여줍니다 |
| 지구-태양 고정 뷰 | 지구와 태양을 고정한 회합주기 좌표계에서 대상 행성의 움직임을 보여줍니다 |
| 지구 중심 뷰 | 지구를 중심으로 태양과 대상 행성의 겉보기 운동을 보여줍니다 |
| 천구 뷰 | 하늘에서 행성이 별들 사이를 어떻게 이동하는지 보여줍니다 |

### 컨트롤 버튼

| 버튼 | 기능 |
|------|------|
| 재생 | 애니메이션을 시작합니다 |
| 일시정지 | 현재 위치에서 애니메이션을 멈춥니다 |
| 정지 | 애니메이션을 멈추고 처음 위치로 돌아갑니다 |

### 슬라이더 사용법

- 슬라이더를 드래그하여 원하는 시점의 행성 위치를 확인할 수 있습니다
- 역행이 일어나는 구간을 직접 찾아볼 수 있습니다

## 외행성 (화성) 시뮬레이션

### 주요 위치

| 위치 | 설명 |
|------|------|
| 합 (Conjunction) | 태양-화성-지구가 일직선, 화성이 태양 뒤에 위치 |
| 충 (Opposition) | 태양-지구-화성이 일직선, 화성이 태양 반대편에 위치 **(역행 발생)** |

### 학습 포인트

- **순행 (Prograde)**: 화성이 별들 사이를 **서 → 동** 방향으로 이동
- **역행 (Retrograde)**: 화성이 별들 사이를 **동 → 서** 방향으로 이동
- **역행의 원인**: 지구가 화성보다 빠르게 공전하므로, 지구가 화성을 추월할 때(충 근처) 화성이 뒤로 움직이는 것처럼 보임

### 기술 정보

- 화성 공전주기: 687일
- 지구 공전주기: 365.25일
- 회합주기: 약 780일 (약 2년 2개월)

## 내행성 (금성) 시뮬레이션

### 주요 위치

| 위치 | 설명 |
|------|------|
| 외합 (Superior Conjunction) | 금성이 태양 뒤에 위치 |
| 동방 최대이각 (Greatest Eastern Elongation) | 금성이 태양 동쪽으로 최대 약 47° 떨어짐 → **저녁별**, 역행 시작 부근 |
| 내합 (Inferior Conjunction) | 금성이 태양과 지구 사이에 위치 → 역행 중간 지점 |
| 서방 최대이각 (Greatest Western Elongation) | 금성이 태양 서쪽으로 최대 약 47° 떨어짐 → **새벽별**, 역행 종료 부근 |

### 위치 사이클

```
외합 → 동방 최대이각(저녁별) → [역행 시작] → 내합 → [역행 종료] → 서방 최대이각(새벽별) → 외합
```

### 역행 구간

**동방 최대이각 부근 → 내합 → 서방 최대이각 부근** (약 42일간, 내합 전후 각 21일)

### 학습 포인트

- **순행 (Prograde)**: 금성이 별들 사이를 **서 → 동** 방향으로 이동
- **역행 (Retrograde)**: 금성이 별들 사이를 **동 → 서** 방향으로 이동
- **역행의 원인**: 금성이 지구보다 빠르게 공전하므로, 내합 근처에서 금성이 지구를 추월할 때 금성이 뒤로 움직이는 것처럼 보임
- **최대 이각**: 내행성은 태양에서 일정 각도 이상 떨어지지 않으므로 새벽이나 저녁에만 관측 가능

### 기술 정보

- 금성 공전주기: 225일
- 지구 공전주기: 365.25일
- 회합주기: 약 584일 (약 1년 7개월)
- 최대 이각: 약 47°

## 수업 활용 예시

1. **역행 구간 찾기**: 슬라이더를 움직여 역행이 시작되고 끝나는 지점을 찾아보세요
2. **천구 궤적 관찰**: 재생 버튼을 눌러 행성이 하늘에서 그리는 궤적을 확인하세요
3. **두 뷰 비교**: 태양 중심 뷰와 지구 중심 뷰를 비교하여 왜 역행이 발생하는지 이해하세요
4. **외행성 vs 내행성**: 두 시뮬레이션을 비교하여 역행 발생 조건의 차이를 이해하세요
